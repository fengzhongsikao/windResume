# 简历生成器 - 支付宝小程序

一款在支付宝小程序端运行的简历制作工具，提供丰富的模板和模块化编辑能力，支持导出 PDF。

## 功能特性

- **模块化编辑** — 支持求职意向、教育经历、实习经历、工作经历、项目经历、技能特长、兴趣爱好、自我评价等 8 种模块，自由组合
- **10 套精美模板** — 涵盖经典蓝、简约白、商务黑、优雅紫、清新绿、暖橙、科技灰、文艺风、极简风、双栏布局等多种风格
- **PDF 导出** — 基于 Canvas 渲染简历并生成 PDF，无需第三方库
- **本地存储** — 使用 zustand 管理状态，数据持久化保存在本地

## 快速开始

### 环境要求

- 支付宝小程序开发者工具
- Node.js

### 安装依赖

```bash
npm install
# 或
pnpm install
```

### 开发调试

在支付宝小程序开发者工具中打开项目根目录，即可预览。

## 使用指南

### 1. 制作简历

在首页点击「制作简历」，进入编辑页。可以添加以下模块：

| 模块 | 说明 |
|------|------|
| 求职意向 | 期望职位、城市、薪资、工作性质 |
| 教育经历 | 学校、专业、学历、时间、描述 |
| 实习经历 | 公司、职位、时间、描述 |
| 工作经历 | 公司、职位、时间、描述 |
| 项目经历 | 项目名称、角色、时间、描述 |
| 技能特长 | 技能名称及熟练度 |
| 兴趣爱好 | 兴趣爱好标签 |
| 自我评价 | 个人简介/自我描述 |

### 2. 选择模板

在首页点击「简历样式」，可以从 10 套模板中选择喜欢的风格。

### 3. 预览 & 导出

在首页点击「我的简历」区域进入预览页，查看简历效果，并导出为 PDF。

## 技术栈

- **框架** — 支付宝小程序
- **语言** — TypeScript
- **样式** — Less
- **UI 组件** — antd-mini
- **状态管理** — zustand
- **PDF 生成** — 自实现 PDF 构建（基于 Canvas 渲染 + JPEG 封装）

## 上传部署

在支付宝小程序开发者工具中上传代码时，项目已开启编译转换，确保 `node_modules` 中的依赖能被正确处理：

```json
{
  "compileOptions": {
    "enableNodeModuleBabelTransform": true,
    "transpile": true
  }
}
```
